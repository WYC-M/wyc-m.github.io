# C++ 学习笔记 - 1 注释

<!--more-->

## 一 注释的作用
对代码进行一些**说明和解释**，方便理解代码

## 二 注释的语法

### 1 单行注释
语法

    // 注释内容

示例

    // 这是一个注释

### 2 多行注释
语法

    /* 注释内容 */

示例  

    /* 这是一个注释
       这是一个注释 */
